## 什么是发现页养号

发现页是小红书的核心内容推荐页面。养薯户会在发现页模拟真实用户行为，自动浏览笔记并进行互动。

## 工作流程

1. 打开小红书发现页 (`xiaohongshu.com/explore`)
2. 启动养号任务
3. 自动滚动加载笔记
4. 根据关键词过滤选择目标笔记
5. 点击进入笔记详情
6. 按概率执行点赞、收藏、评论、关注
7. 返回列表继续下一条

## 配置建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 浏览数量 | 20-30 | 每次任务浏览的笔记数 |
| 浏览间隔 | 5-8秒 | 笔记之间的间隔 |
| 停留时长 | 8-15秒 | 每条笔记的阅读时间 |
| 点赞概率 | 40-60% | 点赞触发概率 |
| 收藏概率 | 20-30% | 收藏触发概率 |
| 评论概率 | 10-20% | 评论触发概率 |

## 注意事项

- 建议每天养号1-2次，每次20-30条
- 避免频率过高，保持自然的互动节奏
- 使用关键词过滤精准定位目标内容
