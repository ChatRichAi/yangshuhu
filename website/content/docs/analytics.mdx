## 数据统计

养薯户记录每次养号操作的详细数据，帮助你评估养号效果。

## 统计维度

- **浏览数** — 总共浏览了多少条笔记
- **点赞数** — 点赞操作次数
- **收藏数** — 收藏操作次数
- **评论数** — 发表评论次数
- **关注数** — 关注用户次数

## 查看方式

### 扩展内查看
打开设置页面 → 「数据统计」标签 → 查看总计和每日明细

### 历史趋势
选择查看最近 7 天、30 天或 90 天的数据趋势。

## 数据导出

在「账户设置」中点击「导出数据」，将所有数据导出为 JSON 文件。
