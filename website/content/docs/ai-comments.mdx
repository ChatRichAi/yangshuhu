## AI 智能评论（专业版功能）

AI 智能评论会读取当前笔记的标题和正文，生成与内容高度相关的个性化评论。

## 评论风格

### 积极正面
AI 会生成鼓励性、正面的评论，适合大多数场景。

### 好奇提问
生成与内容相关的问题，提升互动率和回复概率。

### 分享经验
模拟分享类似经历的评论，增加评论深度。

### 人设评论
根据你设定的角色人设生成评论，保持风格一致性。

### 随机混合
随机选择以上任一风格，使评论更加多样化。

## 配置方法

1. 打开设置页面的「养号规则」
2. 开启「AI智能评论」开关
3. 选择评论风格
4. 确保已在账户设置中配置 API Key

## 使用限制

- 需要专业版或团队版订阅
- 生成速度取决于网络和 API 响应
- 如果 AI 生成失败，会自动回退到自定义评论库
