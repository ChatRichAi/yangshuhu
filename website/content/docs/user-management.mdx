## 账户管理

### 登录方式
- 邮箱 + 密码登录
- 魔法链接（无密码登录）
- Google 账号登录
- GitHub 账号登录

### API Key
每个账户都有一个唯一的 API Key，用于 Chrome 扩展与服务器通信。

在设置页面的「账户设置」中可以查看和重新生成 API Key。

## 订阅管理

### 套餐切换
可以随时在账户设置中升级或降级套餐，费用按比例计算。

### 取消订阅
在账户设置中取消订阅。取消后，当前计费周期结束前仍可使用付费功能。

## 多账号管理（团队版）

团队版支持管理最多 10 个小红书账号，适合运营团队使用。
