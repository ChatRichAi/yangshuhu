## 主页信息流养号

养薯户支持在小红书主页信息流中自动浏览和互动，帮助你的账号建立更精准的兴趣标签。

## 使用方法

1. 打开目标用户的主页
2. 启动养号任务
3. 养薯户会自动浏览主页的笔记列表
4. 按设定规则进行点赞和收藏

## 适用场景

- **建立兴趣标签**：浏览同领域 KOL 的主页，帮助算法了解你的兴趣
- **竞品分析**：自动浏览竞争对手的内容
- **粉丝互动**：浏览粉丝主页，增加互动

## 配置建议

- 主页养号每次浏览 10-15 条即可
- 不建议过于频繁访问同一主页
- 配合发现页养号使用效果更佳
