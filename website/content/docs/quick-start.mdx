## 安装养薯户

### 方式一：Chrome 商店安装（推荐）

1. 打开 Chrome Web Store
2. 搜索「养薯户」
3. 点击「添加到 Chrome」

### 方式二：离线安装

1. 下载最新安装包（v3.0.1）
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 将下载的 `.crx` 文件拖入页面

## 配置养号规则

安装完成后，点击扩展图标进入设置页面：

1. **浏览设置** — 设置每次浏览笔记数量和间隔
2. **互动设置** — 调整点赞、收藏、评论、关注的概率
3. **关键词过滤** — 设置优先浏览和排除的关键词

## 开始养号

配置完成后，有两种方式启动：

- **手动启动**：点击扩展弹窗中的「开始养号」按钮
- **定时执行**：在设置页面配置定时任务，自动执行

## 查看数据

养号过程中和完成后，可以在设置页面的「数据统计」标签查看：

- 今日互动数据
- 历史趋势图表
- 操作日志记录
