## 常见问题

### 扩展无法启动任务
1. 确保已打开小红书页面
2. 检查是否已登录小红书
3. 尝试刷新页面后重试
4. 检查扩展权限是否正确授予

### AI 评论功能不可用
1. 确认已升级到专业版
2. 检查 API Key 是否正确配置
3. 确认网络可以正常访问

### 自动化操作被中断
1. 确保浏览器标签页保持活跃
2. 避免在自动化过程中手动操作页面
3. 检查网络连接是否稳定

### 数据不显示
1. 刷新设置页面
2. 检查浏览器存储空间是否已满
3. 尝试清除数据后重新开始
