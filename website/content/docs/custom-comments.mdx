## 自定义评论库

自定义评论库允许你预设一批评论模板，养号时随机选取使用。这比固定评论更自然，也不需要消耗 AI 额度。

## 设置方法

1. 打开扩展设置页面
2. 进入「养号规则」标签
3. 找到「自定义评论库」区域
4. 添加你的评论模板

## 评论模板技巧

### 通用型评论
```
写得真好，学到了！
感谢分享，收藏了
太棒了，期待更多内容
好有用的分享！
```

### 提问型评论
```
请问这个怎么操作呀？
想问一下具体的步骤是什么
这个在哪里可以买到呢？
```

### 分享型评论
```
我也试过，确实不错
之前也遇到过一样的问题
跟我的经验一样，很认同
```

## 注意事项

- 建议准备 20-30 条不同的评论
- 评论要自然、口语化
- 避免敏感词和广告内容
- 定期更新评论库保持新鲜感
